الهدف التعليمي

تعلم كيفية استخدام Set و Map في JavaScript لإدارة البيانات بكفاءة، مع التركيز على العمليات الأساسية مثل الإضافة، الحذف، والاسترجاع.

عرض تطبيقي

الشرح البرمجي 
// مثال على Set
const userSet = new Set(["Osama", "Ahmed", "Ali"]);
userSet.add("Reda");
userSet.delete("Ali");

// مثال على Map
const productMap = new Map();
productMap.set("Laptop", { price: 15000, quantity: 3 });
productMap.set("Phone", { price: 8000, quantity: 5 });

console.log([...userSet]);
console.log([...productMap.entries()]);
